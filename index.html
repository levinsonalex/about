Code Examples

Safety in Numbers
2014 - Team Project - Actionscript

A flash game created at the 24 Hours of Good - Google Hackathon. Each team was to code a game that taught its audience about a certain learning objective. Ours was to teach players about the importance of using multiple passwords accross all of your accounts. In our game the player is a hacker attempting to hack into three different victims' accounts. The player must use a password bank that they "bought" off the black market and hack their way into the accounts before time runs out. They start off hacking a Grandma's accounts (very easy since she is not very tech savvy), and progress towards a super hacker. We won 3rd place. Enjoy!

It Will Be Done / This Is The End / Zombie Onslaught

A flash game created after taking my first programming class. The game is composed of 3 Parts - It Will Be Done (Defense Minigame), This Is The End (Platformer), and a bonus mode Zombie Onslaught (FPS). This was my very first foray into the programming world, so it is very unpolished. Enjoy!
